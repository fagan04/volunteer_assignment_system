## How to run:

### Clone the repository:
```bash 
   git clone https://github.com/fagan04/volunteer_assignment_system.git
```
### Navigate to the directory:

```bash 
   cd path/to/your/dir
```
### Build gradle:
```bash 
   ./gradlew build
```
### Run the server:
```bash 
   ./gradlew runServer
```

### Run the GUI:
```bash 
   ./gradlew runGUI
```
